# TheMarceloYT portafolio

<p>
Este proyecto es una aplicación web desarrollada desde cero bajo un enfoque MVC (Model-View-Controller) adaptado, utilizando únicamente HTML, CSS y JavaScript vanilla, sin frameworks externos. La estructura está diseñada de forma modular, escalable y profesional, siguiendo buenas prácticas de organización de carpetas y código limpio, lo que facilita su mantenimiento y extensión.

El repositorio cuenta con licencia, lo que permite a empresas o desarrolladores de conversar sobre la adopción de esta misma estructura para proyectos comerciales o la contratación de servicios para implementar soluciones personalizadas con este modelo:

</p>

<ul>
    <li>Arquitectura MVC nativa implementada en vanilla JS.</li>
    <li>Sistema de enrutamiento propio, permitiendo navegación SPA sin frameworks.</li>
    <li>Organización profesional de carpetas (separación clara de vistas, modelos y controladores).</li>
    <li>Código modular y reutilizable, fácil de escalar.</li>
    <li>Diseño responsive y optimizado con HTML y CSS puros.</li>
    <li>Proyecto bajo licencia con posibilidad de uso o adaptación empresarial.</li>
    <li>Disponibilidad para colaboración o contratación, orientado a negocios que busquen una base sólida y eficiente  para sus aplicaciones web.</li>
    <li>Librerías de CSS y JS propios.</li> 
    <li>Implementación propia de soporte multilenguaje y personalización de temas.</li>
</ul>
